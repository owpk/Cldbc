# Cldbc
## Cldbc – a tiny command line db client

   Structure 
 * `/cldbc.connection` – Contains JDBC driver class descriptions.

 * `/config.xml` – Contains cldbc.connection configurations.
 
    Supports: Postgresql Mysql MongoDb. 
    app uses standard sql;
